Speed up your writing using Text Module Forms.

How it works:

You can create your own text module form by clicking on the menu entry Create Your Form. This allows you to create a clickable form, where each form element is assigned a text module. Use your form by clicking and building a text in such a way. Created forms can be saved to your local computer and later be used with the Use Your Form functionality. The menu also provides you with some form examples.
